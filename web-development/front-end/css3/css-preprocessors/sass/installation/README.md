# Installation SASS

- Install Anywhere ([npm](https://www.npmjs.com/))
    - **npm install -g sass**
- Install on Windows ([Chocolatey](https://chocolatey.org/))
    - **brew install --devel sass/sass/sass**
- Install on Mac OS X ([Homebrew](https://brew.sh/))
    - **brew install --devel sass/sass/sass**
